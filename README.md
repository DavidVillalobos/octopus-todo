# Octopus TODO

TODO list Manager, this desktop application will allow you to create task lists,  
each task can be scheduled and displayed on a dashboard,    
it can also be viewed from a calendar.   

# Build

* Electron  
* Reactjs
* Fundation-sites
* hero-icons

## Install dependencies 

~~~cmd
cd octopus-todo
npm install
~~~
